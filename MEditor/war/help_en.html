Hotkeys

::editation mode

Ctrl+Alt+5: Switch between opened objects.
Ctrl+Alt+PgUp: Select the tab on the left of the currently selected tab.
Ctrl+Alt+PgDn: Select the tab on the right of the currently selected tab.
Ctrl+Alt+C: Close the focused (selected) object.
Ctrl+Alt+P: Display the window for publishing.
Ctrl+Alt+R: Refresh the focused object.
Ctrl+Alt+U: Display the window for entering the new object's PID.
Ctrl+Alt+M: Focus on recently modified tab.
Ctrl+Alt+H: Focuse on "Referenced by" tab.
Ctrl+Alt+D: Display the window for deleting the digital object.
Ctrl+Alt+B: Display the window for entering basic informations into MODS and Dublin Core.
Ctrl+Alt+O: Unlock the digital object.
Ctrl+Alt+Z: Display the window for locking digital object.
Ctrl+Alt+F: Display the window for downloading FOXML and datastreams.
Ctrl+Alt+S: Display the window for saving a working copy on the server.
Ctrl+Alt+:W Display the persistent URLs to the digital object.

::general

Enter: Used for confirmation.
Esc: Close opened pop-up window (e.g. Window for entering PID).
Tab: Navigation across the form fields.
Arrows: Navigation inside the form field or navigation in the scope of the window with previews.
