Hotkeys

Ctrl+Alt+5: Changes focused set of tabs.
Ctrl+Alt+Page Up: Shifts left focused tab in the set of tabs.
Ctrl+Alt+Page Down: Shifts right focused tab in the set of tabs.
Ctrl+Alt+C: Closes focused set of tabs.
Ctrl+Alt+P: Displays window for publishing.
Ctrl+Alt+R: Refreshes focused set of tabs.
Ctrl+Alt+U: Displays window for entering the new object's PID.
Ctrl+Alt+M: Focus on recently modified tab.
Ctrl+Alt+H: Focuses on Referenced by tab.
Ctrl+Alt+D: Displays the window for deleting the digital object.
Ctrl+Alt+B: Displays window for entering basic informations into MODS and Dublin Core.
Ctrl+Alt+O: Unlocks the digital object.
Ctrl+Alt+Z: Displays window for locking digital object.
Ctrl+Alt+F: Displays window for downloading FOXML and datastreams.
Ctrl+Alt+S: Displays window for saving a working copy on the server.
Ctrl+Alt+:W Displays the persistent URLs to the digital object.
Enter: Used for confirmation of any choice.
Esc: Closes opened pop-up window (e.g. Window for entering PID).
